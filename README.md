# 🎂 Modern Age Calculator

A beautiful and responsive age calculator built with HTML, CSS, and JavaScript. Calculate exact age between any two dates with a stunning glassmorphism UI.

## ✨ Features

- 📅 **Single Date Mode** - Calculate age from birth date to today
- 📆 **Date Range Mode** - Calculate exact time between any two dates
- 🎨 **Modern Glassmorphism Design** - Beautiful transparent UI with blur effects
- 🎭 **Animated Background** - Floating gradient circles
- 🎉 **Confetti Celebration** - Fun animation when calculating age
- 📱 **Fully Responsive** - Works perfectly on mobile, tablet, and desktop
- 🌙 **Dark Theme** - Eye-friendly dark interface
- 📊 **Detailed Results** - Shows years, months, days, weeks, total days
- 🎯 **Next Birthday Countdown** - See days until your next birthday
- 📅 **Day of Week** - Know which day you were born

## 🔧 How to Use

### Single Date Mode:
1. Select your date of birth
2. Click "Calculate Age"
3. View your exact age in years, months, and days

### Date Range Mode:
1. Click "Date Range" toggle
2. Select start date and end date
3. Click "Calculate Age"
4. View the exact time difference
